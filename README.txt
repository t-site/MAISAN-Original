こんにちは。はじめましてパソコンです。

これはこどもむけにかかれたコンピューターとっくんプログラムです

つかいかたはAIですが、じつはAIではありません。
コンピューターはこのプログラムではじぶんまるだしなので、
ここできたえないと大人のコンピューターのつかいかたはあぶないです。

コンピューターにいうことをきかせるまえに、
コンピューターとむきあうことをかんがえてみませんか。

いろいろなことがおこるとおもいますが、まえもってたいけんしよう。

ねんのため、つかいかたをせつめいします

じつはLinuxというもののうえでうごきます
Linuxのことはおやたちにせつめいしてもらってください
ふつうのパソコンならCopilotさんのほうがいいでしょう。

端末というプログラムで
sh iu.sh today.txtをおしてEnterしてください
おわりかたは勉強しよう。

today.txtはたんなるぶんですが、かきかえたりけしたりはぜったいにしないこと
べつのぶんをかいてなにかをつかってつぎたしてつかいましょう
べつのぶんのたとえはすでに添付してあります

とくべつなコードブロックがあり、このプログラムのたぐいでしかつかってはなりません

またつかうためのコードをかいたひととのやくそく、
ライセンスについてはひじょうにむずかしいのでそうだんください

today.txtはDynabookむけにちょうせいされています
生のコンピューターはDynabookでべんきょうしたほうがいい
ほかに　あなたのさいしょのいちだい　などとせんでんされているものがいいでしょう
